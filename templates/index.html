Bee-Cryptor

Use the encryption standard of the future that's got everyone buzzing!






What is this?

This is a really silly project by me, Cayden Wright, to show off my Python and Flask skills. This program takes your input, and using the Bee Movie script, replaces each character with a word from the script. You can then send this to your friends, where they can paste it in and press "Decrypt", and get the original text back.

See my blog post for more info!
